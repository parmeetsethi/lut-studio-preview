About

LUT Studio

LUT Studio is a tool for designing bespoke colour lookup tables for the !Camera iOS app. Everything happens in your browser — no upload, no account.

What is a LUT?

A LUT (look-up table) is a colour map. Every possible input colour gets a specific output colour. !Camera applies your LUT in real time as you shoot, so the preview, the histogram, and the saved file all reflect your look without any post-processing.

How to use this app

  • Start with a built-in preset, or drop in a Lightroom .xmp file from the Library tab
  • Adjust using the panel tabs: Tone, WB, Curves, Mixer, Grading
  • Preview against the included reference photos or upload your own
  • Hit Export to download a .cube file; AirDrop it to your iPhone
  • In !Camera: Settings → LUTs → Add LUT

Hidden tricks

  • Tap any colour in the preview to auto-balance white from there
  • Hold the preview (or press Space) to compare with the original
  • 1–6 switch panels; ⌘Z/⇧⌘Z undo/redo
  • Drag the grip between panels to resize; double-tap the handle to reset
  • Double-tap any slider to reset it to zero

Limitations

LUTs can't encode anything that depends on neighbouring pixels — clarity, sharpening, grain, vignette, lens corrections, masking. Those have to live in Lightroom or post-capture. The LUT only handles per-pixel colour transforms.
